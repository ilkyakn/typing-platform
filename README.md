# F Keyboard Typing Platform

A browser-based typing practice platform designed for **Turkish F keyboard** users, especially those preparing for
**zabıt katipliği** and **icra katipliği** typing exams.

The platform focuses on real exam conditions, accuracy, and speed rather than game-style typing tests.

---

Features

- 3-minute exam-based typing system
- Correct word counter
- Real-time typing statistics
- Turkish F keyboard focused typing area
- Word-based accuracy calculation
- Case-insensitive and punctuation-tolerant comparison
- Fullscreen mode
- Exam-oriented and distraction-free interface
- Fully client-side (no backend)

---

Project Structure

index.html  
css/style.css  
js/app.js  
js/stats.js  
js/timer.js  

---

How to Use

1. Open the project folder  
2. Open index.html in your browser  
3. Select a text and start typing  
4. Practice under real exam conditions  

No installation and no dependencies are required.

Türkçe Açıklama

Bu proje, **zabıt katipliği ve icra katipliği** uygulamalı sınavlarına hazırlanan adaylar için geliştirilmiş,
